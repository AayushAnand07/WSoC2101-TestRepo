WSoC2101: Frontend

Hey folks, Welcome to the frontend tasks. We look forward for your creativity and skills, and working together soon.
The evaluation criterion for the listed tasks is as:
- The tasks are listed in 3 levels, starting from the easiest to tougher, to check your skillset.
- Each task has maximum points listed along with it according to its level.
- You need to follow the task description given and design the given tasks with atmost creativity you can show.
- Tasks will solely be judged on the basis of your design, creativity and skills. Not upon any descriptive content displayed.
- Always expect more points, the better you go beyond the required things, the more perfect score you may expect.
- Last, but the most interesting part, if you are done with the required tasks individually, and have carved them to your best, an extra 5 pointer is reserved, if you can make the individual tasks listed accesible with each other using a Navbar where the homepage is task-1.
Note: The extra 5 points are totally assigned for those who feel they have completed the tasks well in advance and can show a higher skill set. Please don't lose your best work on tasks for these points.
Thanks,
Happy Coding! and All the best.

Frontend Tasks

  1. VIT-AP Profile Page
    This task is to create a webpage with the following points to be considered:
    1.) Set the title of the page as "VIT-AP Profile".
    2.) Use a heading VIT-AP (center aligned).
    3.) Set the background as University image. (Can be found in resources folder in the repo)
    4.) A short description of the university. (If you don't find a description feel free to use Lorem ipsum para. There are no points for the content)
    5.) Also add a footer saying "Made by "yourname_reg.no." .
    10 Points
  2. Login/Signup form
    In this task we expect you to create a signin/signup form.
    The necessary points are:
    1.) The first diaplay page should not contain the form directly, there should be a click button for signup / register which takes the user to the form.
    2.) The form should be able to take input from the user, although processing it for anything is not required.
    3.) The input info required is Email, password, and a checkbox for keep me logged in or remember me.
    4.) Add a heading WSoC model sigup / registration form.
    20 Points
  3. Carousel Animation
    A carousel is a set of rotating banners, or a slideshow, that displays on the webpage.
    The necessary points are:
    1.) Add atleast 5 images to the carousel. ( Refer to the resources folder for the images )
    2.) The user should be able to maually change the pictures in the carousel as well as it may change periodically itself.
    3.) Provide a paragraph after the carousel. ( Feel free to use Loren ipsum para if you cannot think of adding any info)
    30 Points
