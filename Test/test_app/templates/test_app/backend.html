Backend Tasks

  1. Notes Adder
    Design an application that gives the user a choice to add a note. The note contains two parts, one is its heading, second a small description of the same.
    Possiblities for improvements/additions are endless, so please don't move far ahead of the required things, but better applications are always appreciated.
    10 Points
  2. Login System
    Design a system that lets the user :
    1.) Register by entering a username and password.
    2.) Login using the username and password used while registration.
    3.) Give an option for reset password.
    20 Points
  3. Login system using Google Id
    This is a continuation of the second task.
    In this task, you are required to register/sign in the user using their gmail id's.
    Remove the change password, and register options, and directly sign them in using their gmail id.
    15 Points
